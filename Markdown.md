## Списки

Что бы добавить нумерацию списков необходимо пункты выделить (*). Пример:
* Элемент 1
* Элемент 2
* Элемент 3

Что бы добавить нумерованные списки необходимо пункты пронумеровать. Пример:
1. Пункт номер один
2. Пункт номер два
3. Пункт номер три
